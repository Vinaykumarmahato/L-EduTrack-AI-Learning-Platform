HTML Introduction

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages. HTML provides the structure and layout of a web page by defining different elements like headings, paragraphs, links, images, and other content.

The Role of HTML

HTML plays a crucial role in web development. It allows developers to structure the content of a webpage and display it in a web browser. Every webpage you visit is created using HTML, and it is one of the most important building blocks of the web.

HTML Elements and Tags

HTML is composed of elements. An element generally consists of an opening tag, content, and a closing tag. Here’s an example of an HTML paragraph:

<p>This is a paragraph of text.</p>

The <p> tag is the opening tag, and </p> is the closing tag. The content between the tags is what gets displayed on the web page.

HTML Syntax

HTML documents follow a very simple syntax. Here's a basic example of an HTML document:

<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Welcome to My First Webpage</h1>
    <p>This is an introductory HTML page.</p>
  </body>
</html>

This structure begins with the <html> element and is followed by the <head> and <body> sections. The <head> contains metadata such as the title, and the <body> contains the content that is visible to the user.

HTML Tags and Attributes

HTML elements are defined using tags. Tags usually come in pairs: an opening tag and a closing tag. Tags can also contain attributes that provide additional information about an element. For example:

<img src="image.jpg" alt="An example image">

The <img> tag is used to embed an image in a webpage. The src attribute defines the image source, and the alt attribute provides a description for accessibility purposes.

HTML Structure

Every HTML document has a structure that is important for the browser to interpret it correctly. The basic structure includes:

  • <!DOCTYPE html> declaration to specify HTML5.
  • <html> as the root element.
  • <head> section for metadata and linking external files.
  • <body> section where visible content is placed.

Conclusion

HTML is essential for building the web. It defines the content structure and lays the foundation for all web pages. The next step is learning how to structure your web page with proper elements, and soon you will be able to create complete web pages from scratch.

HTML Structure

HTML documents follow a very specific structure that helps browsers render the content correctly. Understanding this structure is essential for anyone looking to build web pages. Below, we will dive deep into the components of an HTML document and how they work together.

1.

Every HTML document begins with the <!DOCTYPE html> declaration. This declaration is used to specify the document type and version of HTML that the browser should use. In modern web development, this is always set to html for HTML5 documents. It tells the browser that this document is written in HTML5, which comes with new features and improvements over older versions of HTML.

<!DOCTYPE html>

2. <html> Element

The <html> element is the root element of an HTML document. Everything inside the <html> tag is part of the HTML document. The <html> tag is also where you can define the language of the document using the lang attribute. This is helpful for accessibility and search engine optimization (SEO).

<html lang="en">

3. <head> Section

The <head> section contains meta-information about the HTML document. This includes the document title, links to stylesheets, and scripts. The <head> tag does not contain visible content on the page, but it’s critical for defining how the document behaves, looks, and is interpreted.

Inside the <head>:
  • <title>: Specifies the title of the document, which appears in the browser tab or window.
  • <meta>: Defines metadata such as the character set, author, or viewport settings for mobile devices.
  • <link>: Links to external stylesheets (CSS files).
  • <script>: Includes JavaScript files or inline scripts.
<head>
  <title>My Webpage</title>
  <meta charset="UTF-8">
  <link rel="stylesheet" href="styles.css">
</head>

4. <body> Section

The <body> section contains the visible content of the web page. This is where you place all the elements that the user will interact with, such as headings, paragraphs, images, links, and other HTML components.

Basic Body Structure:
<body>
  <h1>Welcome to My Website</h1>
  <p>This is my first webpage.</p>
</body>

5. Commonly Used HTML Tags in Body

Within the <body> section, you will use several common HTML tags to create the layout and content of your page. Here are a few examples:

  • <h1> to <h6>: These are heading tags, with <h1> being the most important (largest), and <h6> being the least important (smallest).
  • <p>: The paragraph tag, used to define text paragraphs.
  • <a>: The anchor tag, used to create hyperlinks.
  • <img>: The image tag, used to embed images on the page.
  • <ul>, <ol>, <li>: These tags are used for creating unordered or ordered lists.

6. Closing Tags

HTML tags generally come in pairs: an opening tag and a closing tag. The closing tag is similar to the opening tag, but it has a forward slash (/) before the tag name. For example:

<h1>This is a Heading</h1>

The opening tag is <h1>, and the closing tag is </h1>.

7. Nesting HTML Elements

HTML elements can be nested inside one another. This allows you to create complex page structures. For example, a list of items can be nested inside an unordered list (<ul>), which itself is inside a <div> container.

<div>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
  </ul>
</div>

8. HTML Comments

HTML allows you to add comments in the code. These comments are not displayed in the browser but are useful for developers to leave notes or explanations. Comments are written within <!-- and -->.

<!-- This is a comment -->

Conclusion

The structure of an HTML document is straightforward but very important. By using the correct tags and following a well-organized structure, you ensure that your web pages are functional, accessible, and easy to maintain. As you continue learning HTML, you'll become more familiar with how to use these elements effectively to create well-structured and efficient web pages.

HTML Text Elements

HTML provides a variety of elements to format text and add meaning to it...

Paragraphs and Headings

Text in HTML can be formatted using heading tags such as <h1> to <h6> and <p>...

<h1>Main Heading</h1>
<p>This is a paragraph of text.</p>

Headings are used to define titles or sections, while paragraphs are used for the main body text.

Formatting Text

Text can be formatted using <b> for bold, <i> for italic, and <u> for underlined...

Lists

HTML supports both ordered and unordered lists...

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Using the <ul> and <ol> tags, you can create lists of items either numbered or bulleted.

Text Alignment

Text alignment is controlled using the <div> tag or inline CSS...

HTML Links

Links in HTML are created using the <a> tag. The href attribute specifies the URL...

<a href="https://www.example.com">Visit Example</a>

Links can also be used to link to sections within the same page using IDs...

HTML Images

Images in HTML are included using the <img> tag. The src attribute defines the path to the image file...

<img src="image.jpg" alt="Description">

Always use the alt attribute to provide text descriptions for images to improve accessibility.

HTML Forms

Forms are used to collect user input. The <form> element wraps all form elements...

<form action="/submit" method="POST">
  <input type="text" name="username">
  <input type="submit">
</form>

Forms can include input fields, radio buttons, checkboxes, and more...

HTML Tables

Tables in HTML are created using the <table> tag. Rows are defined by <tr>, and columns by <td>...

<table>
  <tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
  </tr>
</table>

HTML Semantic Elements

HTML5 introduced semantic elements like <header>, <footer>, and <article>...

<header>
  <nav>Navigation links here</nav>
</header>
<footer>
  <p>Footer content</p>
</footer>

HTML5 New Features

HTML5 brought several new features like native video and audio support, <video> and <audio>...

<video controls>
  <source src="movie.mp4" type="video/mp4">
</video>
